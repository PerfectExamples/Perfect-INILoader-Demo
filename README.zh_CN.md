本项目展示了如何使用 [Perfect-INIParser Library](https://github.com/PerfectlySoft/Perfect-INIParser) 来读取`.ini`类型配置文件。

本项目采用Swift 3.1工具链中的软件包管理器编译。

## 下载、编译和运行

请使用下列命令行下载、编译和运行
```  
$ git clone https://github.com/PerfectExamples/Perfect-INILoader-Demo.git
$ cd Perfect-INILoader-Demo
$ swift build && swift build && .build/debug/INILoader
```

如果成功的话，应该可以看到本项目根目录下的demo.ini文件的若干变量被读出来。
